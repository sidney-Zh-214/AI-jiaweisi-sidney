## 二、科室基本情况分析

### 2.1 组织架构
- **总人数**：15人
- **分组情况**：
  1. 前期策划组（含2名大学生）
  2. 中期摄影摄像组
  3. 后期配音剪辑组
  4. 公司接待组
- **现状评估**：资源紧张

### 2.2 业务来源和范围
1. **公司内部制片工作**
   - 各部门及子分公司、销售公司的制片需求
   - 宣传片、产品推介片、教学片、纪录片
   - 海外项目制作

2. **活动拍摄与直播**
   - 大中型活动拍摄（如董明珠健康之家活动直播）
   - 产品拍摄及制片
   - 活动直播及连线

3. **委外项目管理**
   - 户外大屏项目
   - 央视、凤凰卫视等合作方片子制作跟进

4. **接待与个人服务**
   - 公司接待
   - 领导个人VCR录制
   - 展厅或外宾来访的拍照录像

- **业务特点**：范围广、需求杂、临时性工作较多

### 2.3 项目周期
- **短周期**：1天（简单会议活动）
- **长周期**：数月（广告片）
- **特点**：项目需求不同，周期长短差异大

### 2.4 临时项目占比
- **占比**：60%-70%是临时接的项目
- **影响**：计划经常被打乱
- **核心痛点**：临时项目占比过高

## 三、主要问题分析

### 3.1 资源排期混乱
#### 现状
- 60%-70%是临时项目，时间紧张
- 人员和设备经常冲突

#### 原因
- 需求方临时提需求
- 领导临时决定
- 影视科只能被动接受

#### 后果
- 计划难以执行
- 项目进度不可控

### 3.2 采购流程繁琐耗时
#### 现状
- 购买道具、食材等要走繁琐采购流程
- 审批流程长，影响拍摄进度

#### 具体场景
1. **食材采购**：审批完成时食材已腐坏
2. **拍摄档期**：因流程延误错过最佳拍摄时间
3. **供应商合作**：流程卡顿影响拍摄进度和质量
4. **风险**：有超期违约的风险

### 3.3 审核没标准，方案来回改
#### 现状
- 领导意见不统一或自我否定
- A领导让改，B领导让改回去
- 子分公司、总部的片都需要多位领导、多个甲方审核

#### 审核问题
1. **多头审核**：多位领导审核
2. **意见冲突**：领导意见相左
3. **反复修改**：根据领导意见多次调整
4. **新领导推翻**：新的领导推翻之前已同意的方案

#### 后果
- 反复修改、返工
- 人员疲惫
- 项目进度延误
- 成本增加

#### 本质问题
- 不是效率问题，是决策权问题
- 涉及审批的领导太多
- 意见相左，没有能拍板做决定的人

### 3.4 委外管理问题
#### 供应商选择
- 靠长期合作或领导指定
- 没有标准化筛选流程

#### 验收流程
- 需需求方和公司领导签字
- 验收标准以领导意见为准
- 验收时领导会推翻之前已同意的想法

#### 隐患
1. **合同风险**：存在违约风险
2. **合作关系**：多次反复调整影响供应商关系
3. **公司信用**：不利于维护公司对外信用与口碑
4. **版权问题**：字体版权问题前置处理不够，收到过律师函

### 3.5 工具不统一
#### 3D模型调拨
- 流程复杂，需多次签字
- 和子公司、供应商等合作方用的三维动画工具不统一

#### 技术问题
- 需要转换工具
- 素材有丢损
- 需要手工转换处理和补图修复
